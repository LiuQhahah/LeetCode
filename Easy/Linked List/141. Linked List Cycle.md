## [141. Linked List Cycle](https://leetcode.com/problems/linked-list-cycle/description/)


### Description

给定一个链表，确定它是否有一个循环。
### Example:


### 思路1 递归


### 代码
```java
   
public class Solution{
    public boolean hasCycle(ListNode head){
        Set<ListNode> nodesSeen = new HashSet<>();
        while(head!= null){
            if(nodesSeen.contains(head)){
                return true;
            }else {
                nodesSeen.add(head);
            }
            head = head.next;
        }
        return false;
    }
}
```
